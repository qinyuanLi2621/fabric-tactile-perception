# fabric-tactile-perception
# Data for: Visual and Tactile Perception of Fabrics

This repository contains the data associated with the study:

> Perceiving the tactile properties of the fabrics using fabric images, fabric videos, and real fabrics

## Data description

- `fabric_ratings_raw.xlsx`  
  Raw participant ratings collected in the experiments.


Each row corresponds to a single fabric. Columns include:

- `participant`: anonymized participant ID  
- `fabric`: fabric ID  
- `condition`: experimental condition  
- `rating`: perceptual rating score  

## Ethical considerations

All data are anonymized and contain no personally identifiable information. Data collection was approved by the relevant institutional ethics committee, and participants provided informed consent.

## Usage

The data may be used for research and educational purposes. Please cite the associated publication if you use these data.
